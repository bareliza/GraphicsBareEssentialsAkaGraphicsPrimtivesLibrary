# GraphicsBareEssentialsAkaGraphicsPrimtivesLibrary

C sharp version

Requirements:

  Microsoft .NET Core:

    sudo apt-get install dotnet-sdk


  Possibly Mono development tools too, including compiler:

    sudo apt-get install mono-devel
    sudo apt-get install mono-mcs
